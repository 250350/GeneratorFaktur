Política de Privacidad

1. Responsable del tratamiento

Titular: Generador de Facturas
Email: contacto@generadordefacturas.es

2. Datos que se recogen

Este sitio web trata datos introducidos por el usuario en formularios, como nombres, direcciones, NIF, datos de facturación y datos de pago.

Estos datos son proporcionados directamente por el usuario y son necesarios para generar documentos de facturación.

3. Finalidad del tratamiento

  • Generación de facturas
  • Generación de documentos PDF
  • Procesamiento temporal en sesión de usuario

4. Conservación de datos

Los datos no se almacenan de forma permanente en bases de datos. Se conservan únicamente durante la sesión activa del usuario.

Los datos pueden ser enviados a un servicio externo de generación de PDF exclusivamente para la creación del documento solicitado.

5. Base legal

El tratamiento de datos se basa en el consentimiento del usuario y en la necesidad de ejecución del servicio solicitado.

6. Cesión de datos

No se ceden datos a terceros con fines comerciales.

Para la prestación del servicio, los datos pueden ser procesados por proveedores tecnológicos que actúan como encargados del tratamiento, incluyendo servicios de generación de documentos PDF y almacenamiento técnico de datos.

Estos servicios están alojados en infraestructura en la nube proporcionada por terceros (incluida la plataforma Render), dentro del Espacio Económico Europeo o en entornos que cumplen con estándares adecuados de protección de datos.

7. Derechos del usuario

El usuario puede solicitar acceso, rectificación o eliminación de datos contactando por email.

8. Seguridad

Se aplican medidas razonables para proteger los datos durante su procesamiento.

9. Cookies y tecnologías similares

Este sitio web no utiliza cookies con fines de seguimiento, publicidad o análisis de comportamiento del usuario.

Únicamente podrán utilizarse cookies técnicas estrictamente necesarias para el funcionamiento básico del servicio y la correcta navegación en la plataforma.

10. Contacto

Email: contacto@generadordefacturas.es
